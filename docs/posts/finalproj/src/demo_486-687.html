Simplify Race + Ethnicity

With these commands, we problematically, but conveniently, reduce the race+eth into 5 categories: - NH White as pop_nhwhiteE - NH Black as pop_nhblackE - Hisp/Latx as pop_hispltxE - NH Asian as pop_nhasianXE - NH Multi/Other as pop_nhotherXE

# Computes the NH Asian Population
bmore_bg_income$pop_nhasianXE <- bmore_bg_income$pop_nhasianE + bmore_bg_income$pop_nhhwnpiE

# Computes the NH "Other" Population
bmore_bg_income$pop_nhotherXE <- bmore_bg_income$pop_nhamindE + bmore_bg_income$pop_nhotherE + bmore_bg_income$pop_nhtwomrE

Water

In order to remove the water to make the geography look good, we have to create an erase function and fix some topology issues with the water shapefile itself.

The water from the census has a lot of different water features. We’re going to speed up the erase by first only keeping water features that have a large area.

# Create an erase function
st_erase <- function(x, y) {
  st_difference(x, st_make_valid(st_union(st_combine(y)))) # clipping x intersection (y)^(complement)
}

# Download the water for Baltimore city and county
bmore_water <- area_water("MD", c(510,005), class = "sf") %>% filter(AWATER > 20000)
Retrieving data for the year 2022
bmore_water <- st_transform(bmore_water, 3857) # Transform to Web Mercator
bmore_water <- st_make_valid(st_buffer(bmore_water, 0)) # Fix topology

# Erase water from joined variable
bmore_bg_income <- st_erase(bmore_bg_income,bmore_water)

Make and subset grid

# Make a grid
bmore_grid <- st_make_grid(bmore_bg_income,
  2 * 1000, # Kms
  crs = 3857,
  what = "polygons",
  square = FALSE # hexagons
)

# To sf
bmore_grid <- st_sf(index = 1:length(lengths(bmore_grid)), bmore_grid) # Add index

#plot(st_geometry(bmore_grid), border="#aaaaaa", lwd = .1)
#plot(st_geometry(bmore_bg_income), add=T, lwd = .1)

bmore_shape <- st_union(bmore_bg_income)

bmore_grid.intersects <- st_intersects(bmore_shape, bmore_grid)
bmore_grid.subset <- bmore_grid[bmore_grid.intersects[[1]],]


plot(st_geometry(bmore_grid.subset), col="red")
